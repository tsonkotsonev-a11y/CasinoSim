Casino API Simulation - Technical Payload

* Clean Architecture and DDD principles applied to ensure maintainability and scalability.
 
* Dependency Injection & IOC utilized.

* Asymmetrical Flow Control: High-frequency betting logic uses boolean returns; administrative actions use exceptions.

* Parallel Stress Testing: Test harness simulates 1,000 concurrent threads to verify atomicity.

* AI-Augmented Test Velocity: Human-led, AI-augmented test development for 100% path coverage of the core domain logic and edge-case validation.

* Usage:

Run Tests: dotnet test

Run Application: dotnet run --project Casino.ConsoleApp (Type exit to shut down).
